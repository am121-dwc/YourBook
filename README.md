# Your Book
​
## サイト概要
### サイトテーマ
- 本好きの方が集うコミュニティサイトです
​
### テーマを選んだ理由
- 自分が本を好きでタグにより新しい本を見つけたりできる機能があれば本が好きな人の一助になるかと考えました。
​
### ターゲットユーザ
- 本が好きで新しい本との出会いを望む老若男女です。
​
### 主な利用シーン
- 本を読んでコメントを残す。
- 本を読んだ記録を残す。

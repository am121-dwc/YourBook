## サイト概要
### サイトテーマ
- 本好きの方が集うコミュニティサイトです
​
### テーマを選んだ理由
- 自分が本を好きで同じ好みの人がどんな本を読んでいるかが気になり、好みが同じような方がどんな本を読んでいるのかを見れるようにフォローできたり、タグにより検索し本を探せるサイトを作りたいと考えたからです。
​
### ターゲットユーザ
- 本が好きで新しい本との出会いを望む老若男女です。
​
### 主な利用シーン
- 本を読んでコメントを残す。
- 同じ好みの人をフォローしその人がどのような本を読んでいるのかを見て面白そうであれば参考にし次に読む本を調べる事です。
